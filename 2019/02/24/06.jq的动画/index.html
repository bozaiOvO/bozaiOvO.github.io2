February 24 2019

  • 隐藏and显示and切换

  • 上卷下拉.
    `javascript
    $(‘button:first’).click(function(){
    $(‘.d1’).slideDown()//下拉
    })
    $(‘button:eq(1)’).click(function(){
    $(‘.d1’).slideUp()//上卷
    })
    $(‘button:last’).click(function(){
    $(‘.d1’).slideToggle()//这个切换动态效果从下至上。竖向动作，slideToggle 通过高度变化来切换所有匹配元素的可见性
    })

+ 淡入淡出
```javascript
$('button:first').click(function(){
    $('.d1').fadeOut()//淡出
})
$('button:eq(1)').click(function(){
    $('.d1').fadeIn()//淡入
})
$('button:eq(2)').click(function(){
    $('.d1').fadeToggle()//切换
})
//但是！！！！如果用这个都是0到1，可不可以自定义到.8或者.5呢？
$('button:last').click(function(){
    $('.d1').fadeTo( duration, opacity ,callback);
    //必需的 duration参数规定效果的时长。它可以取以下值："slow"、"fast" 或毫秒。必需的 opacity 参数将淡入淡出效果设置为给定的不透明度（值介于 0 与 1 之间）。可选的 callback 回调函数。
})
